The zip folder consist of provisioning of Virtual Networks, Vitual Machines, NSG and Azure SQL DB using the Azure Resource Manager Templates.

The ref link used for this assessment; https://learn.microsoft.com/en-us/azure/virtual-network/quick-create-template

These placeholders in each of this deployment templates are filled in with specific values when deploying the template, allowing for customization and flexibility in deploying resources to Azure. This template can be deployed using Azure CLI, PowerShell, or the Azure portal.

Parameter values in each template can be customised as needed when deploying the template.
